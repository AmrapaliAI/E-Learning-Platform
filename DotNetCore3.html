Summary

.NET Core is a free, open-source, cross-platform framework for building modern, scalable, and high-performance applications.

Dot Net Core

.NET Core provides a foundation for modern programming concepts such as multithreading, Windows Forms (via .NET Core Desktop), data types, and LINQ, while also clarifying the difference between tasks and threads.** Threads are the basic unit of execution that allow multiple operations to run concurrently, while tasks are higher-level abstractions built on threads that simplify asynchronous programming. --- Threads in .NET Core A **thread** is a lightweight process that executes code independently. Multithreading allows applications to perform multiple operations simultaneously, improving responsiveness. **Example: Creating a thread in C#
using System;
using System.Threading;
class Program
{
static void PrintNumbers()
{
for (int i = 1; i <= 5; i++)
{
Console.WriteLine("Number: " + i);
Thread.Sleep(500);
}
}
static void Main()
{
Thread t = new Thread(PrintNumbers);
t.Start();
Console.WriteLine("Main thread continues...");
}
}
**Output:**
Main thread continues...
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
Windows Forms in .NET Core Windows Forms is supported in **.NET Core Desktop** for building GUI applications. It allows developers to create interactive applications with buttons, text boxes, and labels. **Example: Simple Windows Form**
using System;
using System.Windows.Forms;
class MyForm : Form
{
public MyForm()
{
Text = "Hello WinForms in .NET Core";
Button btn = new Button { Text = "Click Me", Left = 50, Top = 50 };
btn.Click += (sender, e) => MessageBox.Show("Button Clicked!");
Controls.Add(btn);
}
static void Main()
{
Application.Run(new MyForm());
}
}
Data Types in .NET Core .NET Core supports a wide range of **data types**: - Value types: `int`, `double`, `bool`, `char` - Reference types: `string`, arrays, objects - Nullable types: `int?` These types ensure type safety and efficient memory usage. LINQ (Language Integrated Query) LINQ allows querying collections using a SQL-like syntax. It simplifies working with data in arrays, lists, and databases. **Example: LINQ Query**
using System;
using System.Linq;
class Program
{
static void Main()
{
int[] numbers = { 1, 2, 3, 4, 5, 6 };
var evenNumbers = from n in numbers
where n % 2 == 0
select n;
foreach (var num in evenNumbers)
Console.WriteLine(num);
}
}
**Output:**
2 4 6
Difference Between Task and Thread
- **Thread**: A low-level unit of execution. You manually manage its lifecycle.
- **Task**: A higher-level abstraction built on threads. It represents asynchronous work and integrates with the **Task Parallel Library (TPL)**. Tasks are easier to manage, cancel, and compose compared to raw threads.
**Example: Task vs Thread**
using System; using System.Threading;
using System.Threading.Tasks;
class Program
{
static void Main()
{
// Thread example
Thread t = new Thread(() => Console.WriteLine("Running in a thread"));
t.Start();
// Task example
Task.Run(() => Console.WriteLine("Running in a task"));
} }
In summary, **.NET Core** provides developers with powerful tools: threads for concurrency, Windows Forms for GUI, strong data types for safety, LINQ for data querying, and tasks for simplified asynchronous programming.
