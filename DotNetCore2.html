Dot Net Core

Here a very simple .NET Core console program written in C#. It demonstrates basic input and output, and I'll show you the expected output clearly without using any special characters or symbols. Program (C# in .NET Core)
using System;
namespace SimpleDotNetCoreApp
{
class Program
{
static void Main(string[] args)
{
Console.WriteLine("Welcome to .NET Core Program");
Console.WriteLine("Enter your name:");
string name = Console.ReadLine();
Console.WriteLine("Hello " + name + ", nice to meet you!");
}
}
}
Expected Output
When you run the program, it will look like this in the console:
Welcome to .NET Core Program
Enter your name:
John
Hello John, nice to meet you!

This is the most basic example: it prints a welcome message, asks for your name, and then greets you.
