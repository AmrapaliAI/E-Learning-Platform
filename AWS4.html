Module: AWS Storage Services

This module covers AWS storage solutions for different workloads, from object storage to archival.

1. Amazon S3 (Simple Storage Service)

  • Definition: Object storage for unstructured data.
  • Key Features: Scalability, durability, lifecycle policies, versioning.
  • Use Cases: Backup, static website hosting, big data storage.

2. Amazon EBS (Elastic Block Store)

  • Definition: Block storage volumes for EC2.
  • Key Features: Persistent storage, snapshots, SSD/HDD options.
  • Use Cases: Databases, transactional workloads.

3. Amazon EFS (Elastic File System)

  • Definition: Shared file storage accessible by multiple EC2 instances.
  • Key Features: Scalable, managed NFS file system.
  • Use Cases: Content management, shared storage for apps.

4. Amazon Glacier (S3 Glacier)

  • Definition: Archival storage for long-term backups.
  • Key Features: Low-cost, retrieval options (minutes to hours).
  • Use Cases: Compliance archives, cold data storage.

Module: AWS Networking

Learn how AWS connects resources securely and globally.

1. Amazon VPC (Virtual Private Cloud)

  • Definition: Isolated virtual networks in AWS.
  • Key Features: Subnets, routing tables, security groups, NACLs.
  • Use Cases: Secure multi-tier applications.

2. AWS Route 53

  • Definition: DNS and domain registration service.
  • Key Features: Traffic routing, health checks.
  • Use Cases: Domain hosting, failover routing.

3. AWS VPN

  • Definition: Securely connect on-premises networks to AWS.
  • Key Features: Site-to-site VPN, Direct Connect integration.
  • Use Cases: Hybrid cloud connectivity.

4. Amazon CloudFront

  • Definition: Content delivery network (CDN).
  • Key Features: Edge caching, HTTPS, DDoS protection.
  • Use Cases: Media streaming, global web content delivery.
