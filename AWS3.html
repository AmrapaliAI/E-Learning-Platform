Course Syllabus: AWS Fundamentals

Module: AWS Compute Services

Explore AWS compute services that provide processing power for applications and workloads.

  • Amazon EC2: Virtual servers with customizable resources.
  • AWS Lambda: Serverless compute triggered by events.
  • Amazon Lightsail: Simplified compute for small workloads.
  • Amazon ECS/EKS: Container orchestration (Docker & Kubernetes).
  • AWS Fargate: Serverless container execution.
  1. Launch an EC2 instance.
  2. Deploy a Lambda function.
  3. Set up Lightsail for a blog.
  4. Run containers with ECS/EKS.

Module: AWS Storage Services

Learn how AWS stores and manages data across multiple services.

  • Amazon S3: Object storage.
  • Amazon EBS: Block storage for EC2.
  • Amazon EFS: Shared file storage.
  • Amazon Glacier: Archival storage.
  1. Create an S3 bucket.
  2. Attach an EBS volume.
  3. Share files with EFS.
  4. Archive data in Glacier.

Module: AWS Networking

Understand how AWS connects resources securely and globally.

  • Amazon VPC: Isolated virtual networks.
  • AWS Route 53: DNS and domain registration.
  • AWS VPN: Secure hybrid connectivity.
  • Amazon CloudFront: Content delivery network.
  1. Design a VPC with subnets.
  2. Configure Route 53 DNS.
  3. Set up VPN Gateway.
  4. Distribute content via CloudFront.

Module: AWS Databases

Explore AWS database services for structured and unstructured data.

  • Amazon RDS: Managed relational databases.
  • Amazon DynamoDB: NoSQL database.
  • Amazon Redshift: Data warehousing.
  • Amazon ElastiCache: In-memory caching.
  1. Launch an RDS instance.
  2. Use DynamoDB for NoSQL.
  3. Analyze data with Redshift.
  4. Accelerate queries with ElastiCache.

Module: AWS Security

Learn how AWS secures resources and data.

  • AWS IAM: Identity and access management.
  • AWS Shield: DDoS protection.
  • AWS KMS: Key management service.
  • AWS CloudTrail: Logging and auditing.
  1. Create IAM users and roles.
  2. Enable Shield for protection.
  3. Manage keys with KMS.
  4. Track API calls with CloudTrail.

Module: AWS Monitoring

Monitor and optimize AWS resources.

  • Amazon CloudWatch: Metrics and alarms.
  • AWS Config: Resource configuration tracking.
  • AWS X-Ray: Application tracing.
  1. Set CloudWatch alarms.
  2. Track changes with Config.
  3. Trace requests with X-Ray.

Assessment

Evaluate your AWS knowledge:

  1. Quizzes on AWS services.
  2. Hands-on labs (EC2, S3, IAM).
  3. Capstone Project: Build a cloud-native app using multiple AWS services.

Summary

AWS Fundamentals cover compute, storage, networking, databases, security, and monitoring. Step by step, you learn to deploy, secure, and optimize cloud resources for scalable applications.
